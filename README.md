==============================================================================================================================================

Sandbox for trying out animations.

So far we've got:

* Blink
* Bounce
* Fade In
* Fade Out
* Flip
* Move
* Rotate
* Sequential
* Parallel
* Slide Down
* Slide Up
* Zoom In
* Zoom Out
